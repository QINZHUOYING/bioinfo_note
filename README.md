# 生物信息学学习记录

## 第一堂课笔记：算法与模型的区别

### 算法（Algorithm）
- **定义**：解决问题的明确步骤（如菜谱步骤）  
- **示例**：排序算法（快速排序）、BLAST序列比对算法

### 模型（Model）  
- **定义**：对现实系统的抽象描述（如地图是地理的模型）  
- **示例**：隐马尔可夫模型（HMM）、神经网络模型  

**核心区别**：  
✅ 算法是“怎么做”（How）  
✅ 模型是“是什么”（What）
